## 문제 1  
**Q: Define four symbolic constants that represent integer 25 in decimal, binary, octal, and hexadecimal formats.**  
**A:** 25를 10진수, 2진수, 8진수, 16진수 형식으로 나타내는 4개의 심볼릭 상수를 정의한다.  
```asm
DEC25 = 25  
BIN25 = 11001b  
OCT25 = 31o  
HEX25 = 19h  
```
